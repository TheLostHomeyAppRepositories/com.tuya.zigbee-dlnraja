Installation

Choisissez la méthode d'installation qui vous convient

Homey App Store

Méthode recommandée pour la plupart des utilisateurs.

Ouvrir l'application Homey

Sur votre téléphone ou tablette

Aller dans "Apps" → "Découvrir"

Section des applications disponibles

Rechercher "Tuya Zigbee"

Tapez le nom de l'application

Cliquer sur "Installer"

Suivre l'assistant de configuration

Installation Manuelle

Pour les utilisateurs avancés et développeurs.

# Cloner le dépôt
git clone https://github.com/dlnraja/com.tuya.zigbee.git
cd com.tuya.zigbee

# Installer les dépendances
npm install

# Installer sur Homey
homey app install

# Valider l'application
homey app validate
